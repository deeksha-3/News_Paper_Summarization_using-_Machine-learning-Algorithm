 News Paper text  Summarizer using Machine Learning Algorithm:

A desktop application that uses Machine Learning to summarize text quickly and efficiently. It can process web articles, pasted text, PDFs, and Word documents, making it easy to get concise summaries of any content.

Features:

🌐 URL Summarization – Summarize articles directly from the web.
📝 Text Input – Paste your text and generate summaries instantly.
📄 PDF & Word Support – Upload documents to summarize content.
 Machine Learning Algorithm – Uses TF-IDF and TextRank (PageRank) for extractive summarization.
 Save Summaries – Save the summarized text as a file.
 Dark-themed GUI – User-friendly interface with progress indicators.
 How It Works
 Text Extraction – Retrieves text from the chosen source (URL, PDF, Word, or pasted text).
 Sentence Tokenization – Breaks the text into sentences for processing.
 Vectorization – Converts sentences into TF-IDF vectors to analyze importance.
 Similarity Graph – Creates a graph where sentences are nodes connected by similarity scores.
 TextRank Algorithm – Ranks sentences using PageRank to identify key information.
 Summary Generation – Selects top-ranked sentences to form a coherent summary.
